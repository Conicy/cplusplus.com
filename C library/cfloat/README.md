# \<cfloat\> (float.h)

#### 浮点类型的特性

这个头文件描述了特定系统和编译器实现的浮点类型特性。

一个浮点数由四个元素组成：

- 符号：正或负
- 基底（或基数）：表示不同的数字，可以用单个数表示（二进制用 2，十进制用 10，十六进制用 16，等等）
- 有效数字（或尾数）：一系列上述提到的基底数字。这个序列中数字的个数被称作 _精度_。
- 指数（又称作特性值，或范围数）：表示有效数字的偏移量，通过下面的方式影响值：  
_浮点值_ = _有效数字_ x _基底_<sup>指数</sup>，再加上它的符号。
